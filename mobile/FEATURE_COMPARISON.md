## 技术实现建议

### Compose 动画最佳实践
1. 使用 `animate*AsState` 替代 `InfiniteTransition` 做简单动画
2. 对于列表项动画，使用 `AnimatedVisibility`
3. 合理使用 `Modifier.graphicsLayer()` 做变换动画

### 状态管理建议
1. ViewMode 状态: `remember { mutableStateOf(ViewMode.Table) }`
2. Quiz 状态: 使用 `data class QuizState` 封装
3. 高亮元素: `Set<Int>` 存储 atomicNumber

### 性能优化
1. 使用 `derivedStateOf` 计算衍生状态 (如过滤后的元素列表)
2. Canvas 绘制时使用 `remember` 缓存计算结果
3. 避免在 `@Composable` 中进行重量级计算
